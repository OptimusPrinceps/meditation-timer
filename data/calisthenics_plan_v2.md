# Calisthenics Training Plan (v2)

## Weekly Structure

Three full-body strength sessions (A, B, C) plus one conditioning day. Rotated continuously — no fixed weekday assignments. Rest days inserted around climbing or fatigue.

| Session | Emphasis | Skill |
|---|---|---|
| A | Push | Handstand (short touchpoint) |
| B | Pull | Front lever (short touchpoint) |
| C | Balanced full body | — |
| Conditioning | Core, posterior chain, wrist, mobility | — |

Each strength session is full-body — push, pull, and legs every session, with rotating emphasis. Four main exercises per strength block.

---

## Session A — Full Body, Push Emphasis + Handstand

### Warmup (~6 min)

1. 2-3 min light cardio
2. Wrist circles — 10 each direction
3. Scapular push-ups — 2×8
4. Band pull-aparts — 2×15
5. Band external rotations — 2×10 each side
6. 1 priming set push-ups

### Skill Touchpoint (~5 min) — Handstand

| Exercise | Sets × Time |
|---|---|
| Wall walks | 2×1 |
| Chest-to-wall handstand hold | 3×20-45s |

### Strength

| Exercise | Sets × Reps | Notes |
|---|---|---|
| Pseudo-planche push-ups | 4×6-10 | Progress lean depth — visibly further forward each session you hit 10 clean |
| Pull-ups | 3×6 | |
| Pike push-ups | 3×6-8 | Elevate feet on box once 3×8 clean |
| Bulgarian split squats | 3×8 each leg | 3-second eccentric |

### Cooldown (~3 min)

Pec doorway stretch, wrist prayer stretch, thoracic extension.

---

## Session B — Full Body, Pull Emphasis + Front Lever

### Warmup (~6 min)

1. 2-3 min light cardio
2. Wrist circles — 10 each direction
3. Scapular pull-ups — 2×8
4. Band pull-aparts — 2×15
5. Band external rotations — 2×10 each side
6. 1 priming set inverted rows

### Skill Touchpoint (~6 min) — Front Lever

| Exercise | Sets × Time/Reps | Notes |
|---|---|---|
| German hang | 2×15-20s | Active scap engagement |
| Advanced tuck front lever hold | 3×10-12s | Sub-max (~70-80% of current max hold). 2-min rest between. Stop well short of failure to preserve pull-up performance |

**Max test:** every 3 weeks, replace the working sets with one all-out advanced tuck hold to test progression. Reset the working target to ~75% of new max.

### Strength

| Exercise | Sets × Reps | Notes |
|---|---|---|
| Pull-ups | 4×6 | When 4×8 clean → weighted (5kg backpack) or archer pull-up negatives |
| Archer push-ups | 3×5-6 each side | Most load on extended-arm side |
| Inverted rows (feet elevated) | 3×8-10 | |
| Single-leg RDL | 3×8 each leg | Slow tempo, hamstring stretch focus |

### Cooldown (~3 min)

Passive bar hang 2×30s, lat stretch, child's pose.

## Session C — Full Body, Balanced

### Warmup (~6 min)

1. 2-3 min light cardio
2. Wrist circles — 10 each direction
3. Scapular push-ups + scap pull-ups — 1×8 each
4. Band pull-aparts — 2×15
5. Band external rotations — 2×10 each side
6. 1 priming set push-ups or rows

### Strength

| Exercise | Sets × Reps | Notes |
|---|---|---|
| Pull-ups | 3×6 | |
| Pseudo-planche push-ups | 3×6-8 | |
| Pike push-ups | 3×6-8 | |
| Assisted pistol squats | 3×4-6 each leg | Band or pole assistance |

### Cooldown (~3 min)

Hip flexor stretch, lat stretch, wrist prayer.

---

## Conditioning Day — Core + Posterior + Wrist + Mobility

Target: ~35 min.

### Brief Warmup (~3 min)

Light cardio + wrist circles + cat-cow.

### Core (~12 min)

| Exercise | Sets × Reps |
|---|---|
| Hollow body hold | 3×30-45s |
| Hanging leg raises | 3×6-8 |
| Copenhagen plank | 3×15-25s each side |
| Pallof press (band) | 3×8 each side |

### Posterior Chain (~8 min)

| Exercise | Sets × Reps | Notes |
|---|---|---|
| Nordic curl negatives | 3×3-5 | Feet anchored under sofa or with partner |
| Single-leg glute bridges | 2×10 each leg | 3-second hold at top |

If no Nordic setup viable: replace with slider hamstring curls 3×6 or extra single-leg glute bridge volume.

### Wrist Conditioning (~5 min)

- Wrist push-ups (palms down, rock forward/back) — 2×10
- Wrist push-ups (backs of hands) — 2×10
- Rice bucket grip/extension/rotation — 2-3 min if available; otherwise banded wrist work

### Mobility Cooldown (~7 min)

- Couch stretch (hip flexors) — 1×60s each side
- Standing pike or seated forward fold — 1×60s
- Thoracic foam roll or open book rotations — 2 min
- Pec doorway stretch — 1×30s each side
- Ankle dorsiflexion against wall — 1×30s each side

## Weekly Volume (Approximate)

- Pulling: pull-ups ~10 sets + inverted rows ~3 sets (plus climbing once/week)
- Pushing: pseudo-planche ~7 sets + pike ~6 sets + archer ~3 sets = ~16 sets
- Legs: BSS 3 + RDL 3 + pistols 3 = 9 sets
- Skill: 1 handstand session + 1 front lever session
- Core: ~12 sets (conditioning day)
- Posterior chain: ~6 sets (conditioning day)

---

## Progression Ladders

### Handstand
Wall walks → Chest-to-wall hold (60s) → Freestanding attempts → Freestanding 10s → 30s → Wall HSPU

### Front Lever
Tuck (10s) → Advanced tuck (30s) → Single-leg (10s) → Straddle (10s) → Full (5s+)

### Push-ups
Standard → Pseudo-planche (deep lean) → Planche lean → Tuck planche

### Pull-ups
4×8 clean → Weighted 5kg / Archer pull-up negatives → Archer pull-ups → One-arm pull-up negatives

### Legs
Bulgarian split squats → Assisted pistol → Full pistol → Weighted pistol

### Hamstrings
Single-leg RDL → Nordic curl negatives → Nordic curl partials → Full Nordic curl

---

## Progression Rule

When you can do all prescribed sets at the top of the rep range with clean form for 2 consecutive sessions, progress (harder variation, more reps, or added load). For holds: when target time is hit cleanly for 2 sessions, progress.

If form breaks down at the new level, drop back, hold for 2-3 sessions, then re-attempt.

---

## Key Principles

- **Rest between sets:** 3 minutes for strength work, 2 minutes for skill holds. Take it even when it feels unnecessary.
- **Skill before strength** in A and B. Skill quality drops fast under fatigue.
- **Sleep:** 7-8 hours.
- **Walking:** Daily.
- **Shoulder prehab:** Band pull-aparts, face pulls, Cuban rotations — 5 min daily including rest days.
- **Tendon watch:** Elbows, wrists, anterior shoulders during weeks 3-6. First response to nagging: drop a set off the irritating movement, reassess after a week.
- **First week:** drop the top set off each strength exercise. New patterns after 2.5 months on the old plan — earn the higher load.
- **No dips this block.** Revisit at next reassessment if shoulders continue to feel solid.

## What Changed From v1

| Change | Reason |
|---|---|
| Back to full-body sessions | Better fit for your situation — climbing, skill goals, recovery distribution |
| Skill blocks shortened to ~5-6 min | Front lever de-prioritized; maintenance volume only |
| Conditioning day reintroduced as fixed | Not optional. Core, hamstrings, wrist, and mobility live here |
| Hamstring work added (RDLs in B + Nordic on conditioning day) | Posterior chain gap in v1 |
| Dips not reintroduced this block | Marginal benefit doesn't justify the risk asymmetry; revisit at next reassessment |
| Climbing-weekend constraint removed | A-B-C rotates continuously; rest days inserted as needed |
| Each strength block capped at 4 exercises | Session length needs to stay sustainable |
| Wrist work distributed across all sessions + main block on conditioning day | Was siloed and skipped in v1 |
